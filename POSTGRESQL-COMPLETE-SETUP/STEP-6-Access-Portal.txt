===========================================================
   TROUBLESHOOTING:
===========================================================

"Cannot connect" from other PC:
- Check server window is open
- Check IP address is correct
- Make sure on same WiFi
- Try turning off Windows Firewall temporarily

"Page not loading":
- Refresh the page (F5)
- Check the address is correct
- Make sure no spaces in address

"Database error":
- Check PostgreSQL service is running
- Run: net start postgresql-x64-15
- Check database connection in pgAdmin

"Slow performance":
- PostgreSQL should be faster than SQLite
- Check PostgreSQL service status
- Restart PostgreSQL service if needed
